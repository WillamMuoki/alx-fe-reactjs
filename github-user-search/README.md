 # GitHub User Search App (React + Vite)

This is a React application built with Vite.  
It allows users to search for GitHub profiles using the GitHub API (via Axios).

## Features
- Search GitHub users by username
- View basic profile information (name, avatar, bio, followers, etc.)
- Direct link to GitHub profile
